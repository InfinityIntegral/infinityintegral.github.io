#c SGWInput SGWidget interactive text input #ic SGEXTN/widgets/noninstantiable/sgwinput.h #if SGWWidget #d SGWInput is a SGWidget that provides an input field where the user can enter text. Callback functions are triggered when the text is changed, or when the input field gains or loses focus, allowing the user input to be processed. SGWInput has builtin support for SingCorrect, a SGEXTN module that allows users to enter Unicode characters, especially those used in Mathematics, using LaTeX style commands. Since SGWInput needs to intercept user input, it blocks SGWidgets below from receiving input. SGWInput is not directly instantiable and its instances can only be created using the constructors of its subclasses. #dt
_w Distance The position and size of all SGWidgets are calculated using a _w distance.$A _w distance is made of 2 coordinates, coordinate 1 and coordinate 0, and it is denoted using _w(c1, c0). This is equal to c1 * parent_size + c0 * size_unit pixels, where parent_size is the size of the parent SGWidget and size_unit is 1/24 of the screen width or screen height, whichever is longer. #dt
Theme Colours SGWidgets follow a set of 9 chosen theme colours. Each of these 9 colours have a index number between 0 and 8 which can be used to refer to them. If you want a SGWidget to have a colour that is not a theme colour, you must set it using a SGXColourRGBA. Setting a non theme colour switches the widget to non theme colour mode and ignores previously set theme colours.
