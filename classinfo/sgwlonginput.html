Detailed Description

SGWLongInput is a subclass of SGWInput that allows user to enter multiline text into the input field. Tab and enter keys behave as expected in text editors instead of unfocusing the input field or indicating input completion. SingCorrect is enabled on the input field. Since SGWLongInput needs to receive user input to enter text and trigger callback functions, it blocks SGWidgets below from receiving that user input. The constructor of SGWLongInput does not support directly setting the text in the input field. To do so, use SGWInput::setTextFromString(const QString&) instead.

_w Distance

The position and size of all SGWidgets are calculated using a _w distance.
A _w distance is made of 2 coordinates, coordinate 1 and coordinate 0, and it is denoted using _w(c1, c0). This is equal to c1 * parent_size + c0 * size_unit pixels, where parent_size is the size of the parent SGWidget and size_unit is 1/24 of the screen width or screen height, whichever is longer.

Theme Colours

SGWidgets follow a set of 9 chosen theme colours. Each of these 9 colours have a index number between 0 and 8 which can be used to refer to them. If you want a SGWidget to have a colour that is not a theme colour, you must set it using a SGXColourRGBA. Setting a non theme colour switches the widget to non theme colour mode and ignores previously set theme colours. Setting a theme colour switches the widget to theme colour mode and ignores all previously set non theme colours.

 

SGWLongInput(SGWWidget* parent, void (*validationFunction)(SGWInput*), float x1, float x0, float y1, float y0, float w1, float w0, float h1, float h0, float f1, float f0, float s1, float s0)

This constructor for SGWLongInput creates a SGWLongInput and places it onto the screen.
parent: pointer to parent SGWidget
attachedFunction: callback function activated when the user finishes entering text into the input field and unfocuses it, a callback function attached to an input field must return void and take as argument a pointer to a SGWInput, which indicates the input field that triggered the callback function
x1: 1 coordinate of x coordinate (which is a _w distance) of position
x0: 0 coordinate of x coordinate (which is a _w distance) of position
y1: 1 coordinate of y coordinate (which is a _w distance) of position
y0: 0 coordinate of y coordinate (which is a _w distance) of position
w1: 1 coordinate of width (which is a _w distance)
w0: 0 coordinate of width (which is a _w distance)
h1: 1 coordinate of height (which is a _w distance)
h0: 0 coordinate of height (which is a _w distance)
f1: 1 coordinate of font size (which is a _w distance)
f0: 0 coordinate of font size (which is a _w distance)
s1: 1 coordinate of scrollbar width (which is a _w distance)
s0: 0 coordinate of scrollbar width (which is a _w distance)
