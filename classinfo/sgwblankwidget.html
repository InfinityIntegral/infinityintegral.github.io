Detailed Description

SGWBlankWidget displays a coloured rectangle on the screen. This can be used to make borders in UI to split various options into different sections. It can also be placed behind other UI elements to create border effects and fake backgrounds. SGWBlankWidget does not block user input from reaching widgets below.

_w Distance

The position and size of all SGWidgets are calculated using a _w distance.
A _w distance is made of 2 coordinates, coordinate 1 and coordinate 0, and it is denoted using _w(c1, c0). This is equal to c1 * parent_size + c0 * size_unit pixels, where parent_size is the size of the parent SGWidget and size_unit is 1/24 of the screen width or screen height, whichever is longer.

Theme Colours

SGWidgets follow a set of 9 chosen theme colours. Each of these 9 colours have a index number between 0 and 8 which can be used to refer to them. If you want a SGWidget to have a colour that is not a theme colour, you must set it using a SGXColourRGBA. Setting a non theme colour switches the widget to non theme colour mode and ignores previously set theme colours. Setting a theme colour switches the widget to theme colour mode and ignores all previously set non theme colours.

[[nodiscard]] int getThemeColour(bool* isUsing) const

This function gets the theme colour used for the background of this SGWBlankWidget. The boolean isUsing is set to true if this SGWBlankWidget is in theme colour mode and false if this SGWBlankWidget is in non theme colour mode. If nullptr is passed to this function, no mode checking is done.
 

void setThemeColour(int themeColour)

This function sets the theme colour used for the background of this SGWBlankWidget. It also sets this SGWBlankWidget to theme colour mode.

This function sets the colour used for the background of this SGWBlankWidget. The boolean isUsing is set to false if this SGWBlankWidget is in theme colour mode and true if this SGWBlankWidget is in non theme colour mode. If nullptr is passed to this function, no mode checking is done.
 

void setColour(SGXColourRGBA colour)

This function sets the colour used for the background of this SGWBlankWidget. It also sets this SGWBlankWidget to non theme colour mode.
