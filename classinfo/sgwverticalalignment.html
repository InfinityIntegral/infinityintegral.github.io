Detailed Description

SGWVerticalAlignment is designed to replace Qt::Alignment and Qt::AlignmentFlag, it controls the vertical alignment of text in SGWidgets that display text, such as buttons, labels, and input fields. For horizontal alignment, see SGWHorizontalAlignment instead.

 

Flag

This is the type of the vertical text alignment mode flags.
 

Flag Top

This sets text in SGWidgets to align top.
 

Flag Center

This sets text in SGWidgets to align vertical center.
 

Flag Bottom

This sets text in SGWidgets to align bottom.
